# VISTA Project Page

This directory contains the static project homepage for:

**VISTA: View-Consistent Self-Verified Training for GUI Grounding**

The page is configured through `config.js`, rendered by `static/js/render.js`, and styled by `static/css/index.css`. Paper figures and the local PDF are stored under `static/images/` and `static/pdfs/`.

## Local Preview

Open `index.html` directly in a browser, or serve the directory with any static file server.
